# Process management
- ps aux – list all running processes
- top – Real -time CPU and memory usage
- htop – interactive process monitor
- uptime – showing system load average
- kill PID – stop a process gracefully
- kill -9 PID – force stop a process

# file system 
- ls -l: Lists all files 
- pwd: Prints the current working directory path.
- touch file :create empty file
- cd ..   : Moves one level up in the directory structure.
- mkdir : Creates directories 
- tail : shows end of file 
- head : show begining of file
- cp -r: Copies directories and their contents recursively.
- mv: Moves or renames files and directories.
- rm -rf: Forcefully and recursively removes files/folders.
- df -h: Shows disk space usage across all mounted file systems.

# Networking troubleshooting
- ping <host>:   #test the reachability of a host for example - google.com
- ip addr : show addresses assigned to all network interfaces 
- curl Link : display the HTML source og the Link 
- dig : display DNS lookup
